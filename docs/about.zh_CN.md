# Gmail Filter Manager
Gmail Filter Manager 是一款可让您以可视化方式管理 Gmail 过滤器的 Chrome 扩展程序。相比 Gmail 的标准过滤器功能，它提供了更直观、更高效的过滤器创建与管理体验。

## 工具简介
- 通过扩展程序界面，您可以在 GUI 环境中轻松创建和管理复杂的 Gmail 过滤器  
- 支持拖放操作调整过滤器的优先级  
- 已配置的过滤器可导出为 XML 文件，并通过 Gmail 导入来应用  
  - 过滤器**不会发送到任何外部服务器**，而是**通过浏览器的同步功能与您的 Google 帐号关联并进行存储与同步**
- 配置信息保存在浏览器中，可随时加载使用

## 使用方法
### 创建新过滤器
1. 点击过滤器管理界面左侧面板中的“＋”按钮  
2. 在右侧面板中进行以下设置：  
   - 过滤器名称（设置一个易识别的名称）  
   - 条件（发件人、收件人、主题、包含、不包含等）  
   - 操作（跳过收件箱、标记为已读、添加标签等）  
   - 使用“复制此操作”按钮，可在相同操作基础上快速创建新的过滤器  
3. 使用 “+ AND” 按钮添加多个条件  
4. 使用 “+ OR” 按钮添加新的条件组  
5. 所有更改将自动保存

### 更改过滤器优先级
1. 在左侧面板的过滤器列表中，拖动每个过滤器右侧的“≡≡”图标  
2. 移动到希望的位置  
3. 系统会自动保存更改

### 导出过滤器
1. 点击左侧面板的“保存过滤器”按钮  
2. 系统会下载一个 XML 文件

### 在 Gmail 中应用过滤器
1. 准备导出的 XML 文件  
2. 打开 Gmail → 设置（⚙）→ 查看所有设置  
3. 选择“过滤器和封锁的地址”选项卡  
4. 点击“导入过滤器”  
5. 选择导出的 XML 文件并点击“打开文件”  
6. 选择要应用的过滤器并点击“创建过滤器”

### 导入过滤器
1. 点击左侧面板的“加载过滤器”按钮  
2. 选择要导入的 XML 文件  
3. 选择是合并现有过滤器还是全部替换

## 注意事项
- 过滤器数据保存在浏览器中，并可**通过 Google 帐号进行同步（chrome.storage.sync）**  
- 数据**不会发送到开发者的外部服务器**  
- 如果您设置了大量复杂的过滤条件，建议定期导出备份文件以防丢失

## 故障排除
- **问：无法成功导入过滤器？**  
  **答：** 请确认 XML 文件格式正确，仅支持 Gmail 格式的 XML 文件。

- **问：过滤器未保存？**  
  **答：** 可能是浏览器存储空间已满。请删除不必要的数据或导出过滤器以备份。

- **问：Gmail 中未显示按钮？**  
  **答：** 请尝试刷新页面，并确保扩展程序已启用。
