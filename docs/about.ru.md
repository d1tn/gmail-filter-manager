# Gmail Filter Manager
Gmail Filter Manager — это расширение для Chrome, которое позволяет визуально управлять фильтрами Gmail. Оно делает создание и редактирование фильтров более удобным и понятным, чем стандартные средства Gmail.

## Что это за инструмент?
- Инструмент с графическим интерфейсом для создания и управления сложными фильтрами Gmail
- Возможность изменять приоритет фильтров с помощью перетаскивания
- Экспорт настроенных фильтров в XML и их импорт в Gmail  
  - Фильтры **не отправляются на внешние серверы**; они **сохраняются и синхронизируются через ваш аккаунт Google с использованием функции синхронизации браузера**
- Настройки сохраняются в браузере

## Как использовать
### Создать новый фильтр
1. Нажмите кнопку «＋» в левом поле
2. В правой панели настройте:
   - Имя фильтра
   - Условия (От, Кому, Тема, Содержит, Не содержит и т. д.)
   - Действия (Пропустить входящие, Пометить как прочитанное, Присвоить метку и т. д.)
   - Кнопка «Дублировать это действие» создаёт новый фильтр с теми же параметрами
3. Используйте «+ AND» для добавления нескольких условий
4. Используйте «+ OR» для добавления другой группы условий
5. Изменения сохраняются автоматически

### Изменить приоритет фильтра
1. Перетащите значок «≡≡» рядом с фильтром в списке
2. Переместите на нужную позицию
3. Изменения сохраняются автоматически

### Экспорт фильтров
1. Нажмите «Сохранить фильтры»
2. Файл XML будет загружен

### Применить фильтры в Gmail
1. Подготовьте экспортированный XML-файл
2. Gmail → Настройки (⚙) → Все настройки
3. Вкладка «Фильтры и заблокированные адреса»
4. Нажмите «Импортировать фильтры»
5. Выберите XML-файл и нажмите «Открыть»
6. Выберите фильтры и нажмите «Создать фильтры»

### Импорт фильтров
1. Нажмите «Загрузить фильтры»
2. Выберите XML-файл
3. Выберите — объединить или заменить существующие фильтры

## Примечания
- Фильтры сохраняются в браузере и могут **синхронизироваться с аккаунтом Google**
- Данные **не отправляются на внешние серверы**
- При большом количестве фильтров рекомендуется регулярно экспортировать копию

## Устранение неполадок
- **В: Фильтр не импортируется.**  
  **О:** Убедитесь, что XML-файл имеет правильный формат Gmail.

- **В: Изменения не сохраняются.**  
  **О:** Возможно, достигнут предел хранилища браузера. Удалите ненужные данные или экспортируйте фильтры.

- **В: Кнопки не отображаются в Gmail.**  
  **О:** Обновите страницу и убедитесь, что расширение включено.
